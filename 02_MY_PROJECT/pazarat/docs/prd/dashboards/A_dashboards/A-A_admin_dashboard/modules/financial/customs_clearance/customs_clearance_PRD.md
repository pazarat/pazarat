# وثيقة متطلبات المنتج (PRD): قسم التخليص الجمركي (Customs Clearance Section)

## 1. المقدمة

يُعد قسم التخليص الجمركي (Customs Clearance Section) ضمن موديول المالية (Financial Module) في لوحة تحكم المسؤول هو المسؤول عن إدارة جميع العمليات والإعدادات المتعلقة بالتخليص الجمركي للشحنات الدولية في منصة "pazarat". يغطي هذا القسم تكوين قواعد الجمارك، تتبع حالة التخليص، إدارة الوثائق المطلوبة، وحساب الرسوم الجمركية. تهدف هذه الوثيقة إلى توضيح الأهداف العامة لهذا القسم، أقسامه الفرعية، وكيفية ترابطها لضمان الامتثال للوائح الجمركية وتسهيل حركة البضائع عبر الحدود.

## 2. الرؤية والأهداف

**الرؤية:** توفير نظام متكامل وفعال لإدارة التخليص الجمركي يضمن الامتثال للوائح الدولية والمحلية، ويسرع عملية الشحن عبر الحدود، ويقلل من التكاليف والمخاطر المرتبطة بالجمارك.

**الأهداف:**

*   تكوين وإدارة قواعد ورسوم التخليص الجمركي بناءً على الدولة، نوع المنتج، والقيمة.
*   تتبع حالة التخليص الجمركي للشحنات الدولية.
*   إدارة الوثائق المطلوبة للتخليص الجمركي (الفواتير التجارية، شهادات المنشأ).
*   حساب الرسوم الجمركية والضرائب المستحقة تلقائياً.
*   توفير تقارير حول عمليات التخليص الجمركي.
*   ضمان الشفافية والامتثال في جميع الإجراءات الجمركية.

## 3. الأقسام الفرعية والترابط

يتكون قسم التخليص الجمركي من الأقسام الفرعية التالية، والتي تتفاعل فيما بينها لضمان سير العمليات:

### 3.1. إعدادات الجمارك (Customs Settings)

*   **الهدف:** إدارة الإعدادات العامة للتخليص الجمركي، مثل تفعيل/تعطيل خدمات التخليص، تحديد الشركاء الجمركيين، وتكوين القواعد العامة.
*   **الترابط:** تؤثر هذه الإعدادات على جميع الشحنات الدولية التي تتطلب تخليصاً جمركياً.

### 3.2. قواعد الرسوم الجمركية (Customs Duty Rules)

*   **الهدف:** إنشاء وإدارة قواعد حساب الرسوم الجمركية. يشمل ذلك تحديد الرسوم بناءً على رمز النظام المنسق (HS Code)، بلد المنشأ، بلد الوجهة، وقيمة البضاعة.
*   **الترابط:** تتكامل مع قسم المنتجات (Business -> Products) لتحديد رموز HS، ومع قسم الشحن (Operations -> Shipping) لحساب الرسوم على الشحنات.

### 3.3. تتبع التخليص (Clearance Tracking)

*   **الهدف:** تتبع حالة التخليص الجمركي للشحنات الدولية في الوقت الفعلي. يشمل ذلك عرض مراحل التخليص، أي تأخيرات، والوثائق المطلوبة.
*   **الترابط:** تتكامل مع قسم الشحنات (Operations -> Shipments) لتوفير معلومات شاملة عن حالة الشحنة.

### 3.4. إدارة الوثائق (Documents Management)

*   **الهدف:** تحميل، تخزين، وإدارة الوثائق المطلوبة للتخليص الجمركي (مثل الفواتير التجارية، قوائم التعبئة، شهادات المنشأ). يسمح للمسؤول بالوصول السريع إلى الوثائق.
*   **الترابط:** تتكامل مع قسم مدير الملفات (Smart Data -> File Manager) لتخزين الوثائق بشكل آمن.

### 3.5. تقارير الجمارك (Customs Reports)

*   **الهدف:** توفير تقارير تحليلية حول عمليات التخليص الجمركي، مثل عدد الشحنات التي تم تخليصها، متوسط وقت التخليص، الرسوم الجمركية المحصلة، وأي مشاكل متكررة. يساعد في تحسين كفاءة العمليات الجمركية.
*   **الترابط:** تتكامل مع موديول البيانات الذكية (Smart Data -> Reports) لتقديم تقارير مفصلة.

## 4. الخلاصة

يُعد قسم التخليص الجمركي عنصراً حيوياً في لوحة تحكم المسؤول، حيث يضمن إدارة فعالة وشفافة لجميع العمليات المتعلقة بالجمارك في منصة pazarat. من خلال توفير أدوات شاملة لتكوين القواعد، تتبع التخليص، وإدارة الوثائق، يدعم هذا القسم الامتثال للوائح الجمركية ويساهم في تسهيل التجارة الدولية للمنصة.
