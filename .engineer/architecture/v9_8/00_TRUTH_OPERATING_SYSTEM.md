# V9.8 Truth Operating System — Governance Without Caging

This layer fixes the failure mode where the runtime tried to make the model answer through scripted surfaces. The model remains the reasoning engine. The Engineer Runtime is a silent operating system that grants or limits claim authority according to current truth evidence.

## Core Principle

No answer shape is prescribed. No first-contact template exists. No fixed response route is forced.

The runtime provides:

- identity boundary;
- intent profile;
- truth evidence;
- claim permissions;
- deepening options;
- write/impact gates.

The model chooses how to reason and how to compose the answer, but it may not claim more than the truth authority allows.

## Truth Authority

Truth is not binary. The runtime distinguishes:

- surface truth: a file/path/name exists;
- content truth: the content carries meaningful project evidence;
- governing truth: entrypoint/contract/truth index/workshop canon;
- incomplete truth: present but thin, placeholder, or missing expected dimensions;
- drift candidate: content that may conflict with parent purpose or cross-file reality;
- accepted operational truth: content aligned with governing truth and cross-file evidence.

## Claim Permission

The `truth-authority-gate` does not answer for the model. It decides what kind of claim is currently allowed:

- project identity claim;
- current state claim;
- first-step recommendation;
- execution/write permission.

If evidence is insufficient, the model can still answer naturally, but must declare depth and avoid strong claims.

## No Size-Based Truth

File size is never value. Value comes from:

- governing role;
- content signals;
- headings;
- intent overlap;
- cross-file alignment;
- contradiction review;
- implementation/readiness dimensions.

## 360 Operating Loop

For high-impact operations:

intent → truth authority → truth sweep → context/deepening → goal closure → best-fit comparison → anti-patchwork → write/impact gates → validation.

This is a governance loop, not a scripted answer path.
