# Standards Alignment

This framework is inspired by:

- ISO/IEC/IEEE 42010: architecture descriptions through stakeholders, concerns, views, and viewpoints.
- ISO/IEC/IEEE 15288: system life cycle processes.
- NIST AI RMF: govern, map, measure, manage risk.
- Modern agent engineering practice: routing, tools, state, guardrails, tracing, context engineering, and composable workflows.

The framework adapts these ideas to a repository-native project engineer, not a SaaS platform.
