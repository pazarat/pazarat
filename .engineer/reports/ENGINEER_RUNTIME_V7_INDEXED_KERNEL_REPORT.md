# Engineer Runtime v7.0 — Indexed Neural Operating Kernel

v7 is a structural upgrade, not another gate layer. It moves broad repository touching
out of every request and into a materialized kernel index.

## Core change

- `build-index`: precomputes file fingerprints, value signals, high-value files, and relationship edges.
- `plug-fast`: default entry; compact and fast.
- `plug-deep`: selective deep reads for high-impact work.
- `operation-mode`: decides evidence budget and response rhythm.
- `golden-suite`: protects behavior quality.

## Preserved from v6

- workshop universal lens
- central plug
- neural reflexes
- truth hierarchy
- incomplete truth maturity
- best-fit discipline
- no project-specific runtime coupling

## Architectural decision

The index is operational memory, not truth. Current files still outrank any generated index.
