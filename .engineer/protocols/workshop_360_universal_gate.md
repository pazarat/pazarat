# Workshop 360 Universal Gate

The workshop is the first silent knowledge gate after the central plug.

It provides general method and discipline canons for any project type. The engineer uses it to ask:
- what kind of project is this?
- what maturity dimensions apply?
- what local standards are missing?
- what truth is incomplete?
- what execution gates are required?

The workshop never speaks over active project truth. It pressures the project to build or repair local truth before execution.
