# Execution Gates and No Guessing Canon

Execution is not allowed when the required owner truth is missing, conflicting, or incomplete.

## Required Before Execution

- route and intent classification,
- active project truth anchor,
- workshop maturity lens,
- owner file discovery,
- duplicate and impact checks for writing,
- incomplete truth classification,
- local standard or decision for implementation details,
- claim check after output.

## Forbidden

- executing from a folder name,
- executing from a generic best practice alone,
- treating workshop canon as local project truth without specialization,
- writing a parallel concept without owner search,
- claiming absence without a zero-result scan.
